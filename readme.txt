# OnlineChamp Theme Extender Plugin

This plugin is designed to enhance and extend the functionality of WordPress themes developed by OnlineChamp, providing optimizations for your website.

---

## Requirements

- **WordPress Version**: 5.6 or higher
- **PHP Version**: 7.4 or higher
- **OnlineChamp Theme**: Ensure your theme is an official OnlineChamp theme compatible with this plugin.

---

### Will this plugin work with non-OnlineChamp themes?
This plugin is optimized for OnlineChamp themes and may not function as intended with themes from other developers.
